可持久化线段树也叫主席树，主席树的叫法是网络调侃叫法，实际上就应该叫可持久化线段树。

简要说明

可持久化线段树需要将历史数据同时保存，具体做法就是同时维护多颗线段树，每一颗线段树只跟前一个线段树保持不同。有n个节点的可持久化线段是需要建n个线段树，即每一个新元素都要建一个线段树。此为可持久化线段树的基本定义。

观察可知，每棵树跟前一棵树只有部分不同，他们是一个从根节点到叶节点的链。

从空间上来说每一个新节点都建立一个新的线段树有点浪费，因此优化过的线段树主要保存部分内容，通过树之间的联系来维持一个线段树。

主要应用

可持久化线段树可以处理的经典问题是区间第K大/小问题。

线段树相减

为了得到区间的线段树，可以使用前缀和的思想，区间包含的元素等于区间减去

空间优化

每棵树只修改对应的链上节点，然后被修改的节点指向原始空树上的其他子节点，这样的一颗优化主席树在逻辑上就完整了。

代码实现

#include <bits/stdc++.h>

using namespace std;

const int N = 2e5+10;

int cnt = 0;
// root[i]表示第i颗线段树的根节点编号
int a[N],b[N],root[N];

struct{
    //sum是节点的权值
    int l,r,sum;
}tree[N<<5];

// 用来建一颗空树，但是不需要真的执行
int build(int pl,int pr){
    int rt = ++cnt;
    tree[rt].sum = 0;
    int mid = (pl+pr)>>1;
    if(pl < pr){
        tree[rt].l = build(pl,mid);
        tree[rt].r = build (mid+1,pr);
    }
    return rt;
}

// 添加一颗新树
// pre 前一棵树
int update(int pre,int pl,int pr,int x){
    //获取当前树的根
    int rt = ++cnt;
    // 新树的当前节点指向前一棵树相同节点的左右儿子
    tree[rt].l = tree[pre].l;
    tree[rt].r = tree[pre].r;
    // 因为新增了一个节点，因此当前节点的sum要+1
    tree[rt].sum = tree[pre].sum+1;
    int mid = (pl+pr)>>1;

    //向下递归建不超过lg(n)个节点
    if(pl < pr){
        // 根据x的值决定在左右那个分支继续更新
        if(x <= mid) {
            tree[rt].l = update(tree[pre].l,pl,mid,x);
        }
        else {
            tree[rt].r = update(tree[pre].r,mid+1,pr,x);
        }
    }
    return rt;
}

// 查找[u,v]区间的第k大
int query(int u,int v,int pl,int pr,int k){
    if(pl == pr) return pl;

    // 线段树相减
    int x = tree[tree[v].l].sum - tree[tree[u].l].sum;

    int mid = (pl+pr)>>1;
    //左二子的数量大于k时就证明做左子树中
    if(x>=k){
        return query(tree[u].l,tree[v].l,pl,mid,k);
    }
    else {

        //否则就再右儿子树中，这里还要减去x，在子树中药去除左树。
        return query(tree[u].r,tree[v].r,mid+1,pr,k-x);
    }
}

int main(){
    int n,m;cin>>n>>m;
    for(int i=1;i<=n;i++) {
        cin>>a[i];
        b[i] = a[i];
    }
    // 离散化
    sort(b+1,b+1+n);
    int size = unique(b+1,b+1+n) - b - 1;
    for(int i = 1;i<=n;i++){
        int x = lower_bound(b+1,b+1+size,a[i])-b;
        root[i] = update(root[i-1],1,size,x);
    }
    while(m--){
        int x,y,k;cin>>x>>y>>k;
        int t = query(root[x-1],root[y],1,size,k);
        cout<<b[t]<<"\n";
    }
}
